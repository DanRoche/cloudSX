Technical prerequisites for online edition of office documents

For firefox
  1. copy the following :
    network.protocol-handler.expose.vnd.sun.star.webdav
  2. in firefox address bar, enter : about:config
  3. in the config page, do contextual-menu/New/Boolean-value
  4. paste in the option name
  5. set the value at false

After this operation, when you will click the first time on the button Edit directly online, firefox will ask you which software you want to use.
You may choose LibreOffice, and memorize this choice, the question will no longer be asked for the following operations.

For Chrome or Edge with windows 7/8/10 and LibreOffice
  1. download and save the following file : wd-lo5-7-8-10.reg
  2. double-click the file in question, this will launch the regedit utility, accept the installation of the registry key

Following this operation, when you click on the button Edit directly online, the browser will open LibreOffice for document edition.

For Windows 7/8/10 and MS-Office
BE CARREFUL : not tested
  1. download and save the following file matching your office version : wd-ms-office2013.reg  wd-ms-office2010.reg
  2. double-click the file in question, this will launch the regedit utility, accept the installation of the registry key

Following this operation, when you click on the button Edit directly online, the browser will open MS-Office for document edition.

For Chrome/Chromium on ubuntu and LibreOffice
  1. download the following file : vnd-sun-star-webdav.desktop
  2. save the file on /usr/share/applications/
  3. execute sudo update-desktop-database

Following this operation, when you click on the button Edit directly online, the browser will open LibreOffice for document edition.
