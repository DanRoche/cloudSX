Prérequis technique pour édition des documents bureautique en ligne

Pour firefox
  1. copiez ce qui suit :
    network.protocol-handler.expose.vnd.sun.star.webdav
  2. dans la barre d'adresse de firefox, tapez : about:config
  3. dans la page de config, faire menu-contextuel/Nouvelle/Valeur-booléenne
  4. collez le nom de l'option
  5. mettez la valeur false

Suite à cette opération, lorsque vous cliquerez la 1er fois sur le bouton Editer directement en ligne, firefox vous demandera avec quel logiciel vous voulez procéder.
vous pouvez choisir LibreOffice, et memoriser ce choix, la question ne vous sera plus posée pour les opérations suivantes.

Pour Chrome ou Edge sous windows 7/8/10 et LibreOffice
  1. télécharger et enregistrer le fichier suivant : wd-lo5-7-8-10.reg
  2. double-cliquez sur le fichier en question, cela va lancer l'utilitaire regedit, acceptez l'installation de la clef de registre

Suite à cette opération, lorsque vous cliquerez sur le bouton Editer directement en ligne, le navigateur ouvrira LibreOffice pour edition du document.

Pour Windows 7/8/10 et MS-Office
ATTENTION : non testé
  1. télécharger et enregistrer le fichier suivant correspondant à votre version d'office: wd-ms-office2013.reg  wd-ms-office2010.reg
  2. double-cliquez sur le fichier en question, cela va lancer l'utilitaire regedit, acceptez l'installation de la clef de registre

Suite à cette opération, lorsque vous cliquerez sur le bouton Editer directement en ligne, le navigateur ouvrira MS-Office pour edition du document.

Pour Chrome/Chromium sous ubuntu et LibreOffice
  1. télécharger le fichier suivant : vnd-sun-star-webdav.desktop
  2. enregistrez le fichier sous /usr/share/applications/
  3. executez sudo update-desktop-database

Suite à cette opération, lorsque vous cliquerez sur le bouton Editer directement en ligne, le navigateur ouvrira LibreOffice pour edition du document.
